### Develop Environment

- Language : Java
- IDE : Intellij IDE
- BuildTool : Gradle 6.4
- Spring Boot 2.0
- Spring Data JPA
- Google Cloud Platform VM
- Maria DB

### Dependency
- spring-starter-security
- spring-starter-mail
- swagger-ui, swagger2
- queryDsl
- lombok
